# マイナンバーPass プライバシーポリシー

最終更新日: 2026年6月26日

マイナンバーPass（以下「本アプリ」）は、マイナンバー実務検定2級・3級の学習機能、広告表示、アプリ内課金、利用状況の把握と品質改善のために、端末上の学習データおよび外部サービスが提供する識別子等を利用する場合があります。

本アプリは、ログインやアカウント作成を必要としません。

## 収集する情報

- 学習履歴、回答履歴、正誤履歴、苦手問題、連続学習日数など、アプリ内で保存される学習データ
- アプリの設定情報
- 広告配信、広告効果測定、不正防止、広告視聴による追加問題解放に必要な情報
- アプリ内課金の購入状態を管理するために必要な情報
- アプリの起動、学習開始、回答、学習完了、模試開始、模試完了などの利用状況に関する集計情報
- クラッシュ、不具合、利用状況の把握に必要な技術情報

本アプリは、氏名、住所、電話番号、メールアドレス、連絡先、正確な位置情報を、アカウント情報として独自に取得しません。

本アプリは、利用者の個人番号、本人確認書類、マイナンバーカード画像、行政手続に必要な個別情報を入力・保存・送信する機能を持ちません。

本アプリは、問題本文、選択肢本文、解説本文、個別の問題ID、個人番号、本人確認書類、カード画像、行政手続情報を利用状況分析に送信しません。

## 利用目的

- 問題演習、復習、学習進捗表示の提供
- 誤答優先学習、苦手問題、連続学習日数などの学習機能の提供
- 広告表示および広告視聴による追加問題解放
- アプリ内課金の購入状態の確認、復元、管理
- 利用状況の把握、機能改善、不具合調査
- クラッシュやパフォーマンス問題の確認

## 学習データの保存

学習履歴、回答履歴、苦手問題、連続学習日数、広告視聴による追加問題解放の記録などの学習データは、原則として利用者の端末内に保存されます。

本アプリは、学習履歴を管理するための独自アカウントサーバーを運営していません。端末の削除、アプリの削除、端末変更などにより、端末内の学習データが失われる場合があります。

## 広告について

本アプリでは、Google Mobile Ads を利用して、バナー広告、インタースティシャル広告、リワード広告を表示する場合があります。

リワード広告は、無料利用時に分野別の追加問題を解放するために利用します。広告視聴が完了した場合のみ追加問題が解放され、広告を閉じた場合、キャンセルした場合、広告の読み込みや表示に失敗した場合は追加問題は解放されません。

Google Mobile Ads では、広告配信、広告効果測定、不正防止などのために、端末情報、広告識別子、利用状況に関する情報が利用される場合があります。これらの情報の取り扱いは、Google のポリシーに従って管理されます。

## 利用状況分析について

本アプリでは、Firebase Analytics を利用して、アプリの起動、学習開始、回答、学習完了、模試開始、模試完了、購入導線の表示、購入処理、レビュー依頼の対象判定などのイベントを、個人を直接特定しない集計目的で送信する場合があります。

本アプリは、Firebase Authentication、Firestore、Realtime Database、Cloud Functions、Cloud Storage、Remote Config、Crashlytics、Cloud Messaging、App Check を利用しません。

## アプリ内課金について

本アプリでは、非消耗型のアプリ内課金により、全問題の利用、広告非表示、詳細解説、選択肢ごとの解説、苦手問題モード、模擬試験などの機能を提供する場合があります。

購入処理は Apple の仕組みを通じて行われ、購入状態の管理には RevenueCat を利用します。RevenueCat では、購入状態の確認、復元、課金機能の提供に必要な識別子や購入情報が取り扱われる場合があります。

## 第三者サービス

本アプリでは、広告配信、課金管理、利用状況の把握、品質改善のために、以下の第三者サービスを利用する場合があります。各サービスで取り扱われる情報は、それぞれのプライバシーポリシーに従って管理されます。

- Google Mobile Ads
- Firebase Analytics
- RevenueCat
- Apple が提供する App Store、アプリ内課金、クラッシュログ、診断関連サービス

## 外部送信

広告配信、課金管理、利用状況の把握、クラッシュ解析、品質改善のため、第三者サービスに端末情報、広告識別子、購入情報、利用状況に関する集計情報、クラッシュ情報、診断情報などが送信される場合があります。

本アプリは、利用者が入力した氏名、住所、電話番号、メールアドレス、連絡先、正確な位置情報、個人番号、本人確認書類、マイナンバーカード画像、行政手続に必要な個別情報を独自に収集して外部送信する機能を持ちません。
